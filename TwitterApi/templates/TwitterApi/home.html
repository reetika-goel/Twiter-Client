This project is work of "THE SEEKERS" Team that includes Vidhi, Sithara, Pragya and Reetika.

The purpose of this project is to write a client application to implement 8 Twitter API’s. It uses Docker Container framework onto which various components and packages can be deployed. The application is built around a developer's Twitter account. The developer initially requests for the authentication key and token from the Twitter account to work with the APIs. Using the same, the developer can send request to any of the Twitter APIs and receive a response. An HTML page is designed as a part of user interface with CSS styling. When the client clicks on any of these API’s, it redirects to a new page which displays the response to the API Request.
